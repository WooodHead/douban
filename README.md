## 开发

``` bash
# 克隆项目
git clone https://github.com/jasmine-na/douban.git

# 下载依赖 install dependencies
npm install

# 本地开发 开启服务   serve with hot reload at localhost:8080
npm run dev
```
浏览器访问 http://localhost:8080
## 发布
```
# 生产环境
npm run build
```
